# Introduction to Oracle Cloud
>
> We want to implement industry standard based integrated cloud serviuces. All the services to be completely managed and hosted by oracle.
> Our strategy is to use opensource and non proprietery tech. We want to ensure that the integration with on premise environment is seamless. Make oracle cloud productive for customers. Ensure reliability and trustworthiness. Help customer to save money.

## Design goals

* provide broad offerring SaaS, PaaS, IaaS
* built on industry standards
* run transparently on cloud
* self service to users
* flexible upgrades
* enable SOA for on premises integration

## Oracle cloud service portfolio

* IaaS (Infrastructure as a service)
* DaaS (Database as a service)
* Paas (Platform as a service)

Oracle cloud Applications implements

* SaaS
