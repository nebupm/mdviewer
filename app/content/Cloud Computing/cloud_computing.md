# Cloud Computing

    It is a model for enabling ubiquitous, convenient on demand network access to a shared pool of configurable compute resources.
    it provides agility, scalability and reliability

## Characteristics

    Self service
    Rapid elasticity
    Broad network access
    Measured service (helps understand utilization of resources)
    Resource pooling (same resources used by multiple customers)

## Service Models

    SaaS : application are delivered to end users
    PaaS : application development is deployed for developers for them to provide services to their end users.
    IaaS : Computing hardware like compute, network, storage is delivered as service.

## Deployment Models

    Public Cloud
    Private Cloud
    Hybrid Cloud
    Multi cloud

## Cloud computing Challenges

    Interoperability
    Performance
    Reliability (No single point of failure)
    Security (should be super secure)
    Salability (should be scalable)

## Cloud management and Administration

    Provisioning and Orchestration
    Automation
    Security and compliance
    Service request
    Monitoring and logging (Observability)
    Inventory and classification
    Cost management and optimization
